  3. 变量与常量

      * 变量

        * 变量的概念：内存中的一块区域，该区域的数据可以在一定范围内不断变化

        * 变量的构成：数据类型、变量名、变量值

        * 变量的声明：数据类型 变量名;

        * 变量的初始化：数据类型 变量名 = 变量值（第一次赋值即为初始化）;

        * 变量的分类：基本数据据类型变量和引用数据类型变量

        * 注意：变量都有其作用域，只在作用域内有效；同一个作用域内不能声明两个同名的变量；变量赋值时必须满足其数据类型

        * 变量的赋值

          ```java
          public class Main {
              
              public static void main(String[] args) {
                  int a;    //声明一个整数类型变量a
                  int b = 10; //直接在定义变量后面添加 = 10，表示这个变量的初始值为10，这里的10就是一个常量数字
                  int c;
                  c = 12;//也可以先声明再初始化/赋值
                  int d,e;//定义变量d和变量e，用逗号分开就可以
                  int f;
                  int g;//也可以分两次声明
                  final int h = 15;//将变量h定义为一个常量，常量的值只有第一次赋值可以修改，其他任何情况下都不行
                     
              }
          }
          ```

          

      * 常量

        * 有时候我们希望变量的值一直保持不变，我们就可以将其指定为常量在变量前面添加**final关键字，表示这是一个常量,常量的值不允许发生修改**

#### 4.二进制复习

 * 原码：由符号位+真值构成（最高位是符号位，1为负，0为正）
 * 反码
    * 正数的反码是其原码本身，负数的反码是在其原码的基础上，符号位不变，其余各位取反
    * 为了便于计算机计算引入反码
 * 补码
     * 正数的补码是其原码本身，负数的补码是在其原码的基础上，符号位不变，其余各位取反，然后+1；即其反码+1

     * 为了修复0的符号和两个编码的问题，引入补码，还能多表示一个最低数-128

       ```
       一个字节（8bit）表示的范围：
       原码和反码：[-127,127]
       补码：[-128,127]
       ```

#### 6. 类的成员

 * 属性（成员变量）
    * 在类中声明，有默认值
 * 方法
    * 声明格式：返回值类型 方法名（参数列表）{方法体；return 返回值}
    * 返回值：方法体任务完成后返回的结果，如不返回结果，声明方法时返回值类型用void
    * 参数：方法定义时在括号里的是形参，调用方法时传入的是实参
    * 传递机制：Java的传递机制是值传递
 * 代码块
 * 构造器
    * 格式：类名（）{}
    * 应用：可以在这里对属性进行初始化
    * 特点：无返回值，每个类默认有一个空参构造器
 * 静态属性
    * 由关键字``static``关修饰，在类加载的时候就会分配内存，可以通过类名直接访问
    * 类的所有对象共享，某一对象对其改变后，其他对象使用时也是改变后的值
    * 静态变量只会被分配一次内存，即使创建多个对象，这样可以节省内存
 * 静态方法
    * 由关键字``static``修饰，在类加载的时候就会分配内存，可以通过类名直接访问
    * 静态方法无法调用非静态成员，因为类的非静态成员不存在的时候静态方法就已经存在了，此时调用在内存中还不存在的非静态成员，属于非法操作
* 非静态成员
  * 类的对象的属性和方法，即实例的属性和方法
  * 非静态成员属于实例对象，只有在对象实例化之后才存在，需要通过类的实例对象去访问
